# GET /siaa/estado

Devuelve el estado completo del sistema, incluyendo disponibilidad de Ollama, documentos cargados y estadísticas del caché.

## Endpoint

```
GET /siaa/estado
```

## Ejemplo

```bash
curl http://localhost:5000/siaa/estado
```
