# Bienvenido a SIAA

**SIAA** (Sistema Inteligente de Apoyo Administrativo) es el asistente institucional de inteligencia artificial de la **Seccional Bucaramanga de la Rama Judicial de Colombia**. Proporciona respuestas precisas sobre documentos normativos, procedimientos administrativos y requisitos del sistema SIERJU, funcionando de forma completamente local y sin conexión a internet.

| RAM       | 64 GB DDR4 |
| GPU       | ❌ RX 550 (sin soporte ROCm — descartada) |
| SO        | Fedora 43 |
| Modelo IA | qwen2.5:3b (GGUF Q4_K_M, ~1.8 GB en RAM) |

<Info>
SIAA está diseñado para operar **sin GPU**, aprovechando los 64 GB de RAM y el soporte AVX2 del Ryzen 5 PRO 2600 para inferencia eficiente en CPU.
</Info>

## Flujo de una Consulta

```
Usuario (navegador)
      ↓
   Nginx :80
      ↓
SIAA Proxy Flask :5000
  ├── ¿Es saludo?     → Respuesta conversacional directa
  ├── ¿Está en caché? → Respuesta instantánea (~5ms)
  └── Consulta nueva  → RAG Pipeline
        ├── Enrutador TF-IDF (59 docs en RAM)
        ├── Extractor de chunks (ventana deslizante)
        └── Ollama qwen2.5:3b (CPU, AVX2)
              ↓
        Respuesta streaming (SSE)
```
