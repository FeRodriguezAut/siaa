## Instalar Dependencias del Sistema

**Fedora 43 / RHEL (sistema de producción):**

```bash
sudo dnf install libreoffice-headless tesseract tesseract-langpack-spa poppler-utils -y
```

**Debian / Ubuntu:**

```bash
sudo apt install libreoffice-headless tesseract-ocr tesseract-ocr-spa poppler-utils -y
```

---

## Instalar Dependencias Python

```bash
pip install flask flask-cors waitress requests \
            python-docx pandas openpyxl xlrd \
            pymupdf4llm pdf2image pytesseract \
            --break-system-packages
```

<Note>
La opción `--break-system-packages` es necesaria en Fedora 43 y sistemas con entornos Python gestionados externamente.
</Note>

**Dependencias clave:**

| Paquete | Propósito |
|---------|-----------|
| `flask` + `flask-cors` | Servidor web y API REST |
| `waitress` | WSGI de producción (reemplaza a Gunicorn) |
| `python-docx` | Lectura de documentos Word |
| `pymupdf4llm` | Conversión PDF → Markdown (nativo) |
| `pdf2image` + `pytesseract` | OCR para PDFs escaneados |
| `pandas` + `openpyxl` + `xlrd` | Procesamiento de Excel |

## Instalar y Configurar Ollama

```bash
# Instalar Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Descargar el modelo
ollama pull qwen2.5:3b

# Verificar instalación
ollama list
# Debe mostrar: qwen2.5:3b
```
